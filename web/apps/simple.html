I am an app example for navigation options.

To navigate to another app in the same space, change the location path hash of the parent window to reflect the target app navigation path.

Examples:

  • Go to any other app in the same portal

    Another app listed on the home page: onclick="top.window.location.hash='!/app2';"
    Target app definition: 

    				{
				    "label": "Dynamic Resize",
				    "categories": ["Category A", "Category B"],
				    "appUrl": "../app/dynamicsize.html",
				    "url": "app2"
				}

  • Go to any other app in the same portal

    Another app not-listed on the home page: onclick="top.window.location.hash='!/app7';"
    Target app definition: 

    				{
				    "label": "Hidden App",
				    "hpTile": false,
				    "appUrl": "../app_2/hidden-app.html",
				    "url": "app7"
				}

  • Go to another app with query parameters

    onclick="top.window.location.hash='!/app2##a/b/c?p1=1';"

  • Deeplink bookmark

    http[s]://[host]:[port]/[path]/#!/app2##a/b/c?p1=1
